# Building the APK

This document outlines the steps required to build the Android APK for this project.

## Prerequisites

- **Flutter SDK:** You will need to have the Flutter SDK installed on your machine. You can find the installation instructions [here](https://flutter.dev/docs/get-started/install).
- **Android SDK:** You will need to have the Android SDK installed and configured. This is usually done through Android Studio.
- **Java Development Kit (JDK):** You will need to have a JDK installed. This project requires JDK 17.

## Building the APK

Once you have the prerequisites installed and configured, you can build the APK by running the following command in the root of the project:

```bash
flutter build apk
```

The APK will be located in the `build/app/outputs/flutter-apk/` directory.
